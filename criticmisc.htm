Critique inédits des albums + Bonus

 

 

 

Voici mon avis de toutes les chansons inédites des albums, avec 1 ou 2 petits bonus…

 

 

Two of us

sorti en 1998 dans le single Depend on you, two of us est une chanson assez calme, et sympa. Cependant, ce n’est pas vraiment un coup de cœur pour moi, et le niveau reste très très loin des bombes que ayu nous a sorti.

QC : 3/4 ; QR : 2,5/4 ; Arr : 3/5 ; Str 0,5/1 ; Ch : 1/2 ; Par : 1/2 ; CC : 1/2   -> 12/20

 

LOVE ~Destiny~

Une  belle chanson d’amour bien orchestrée, surtout avec du piano. Dommage qu’elle ne figure pas sous cette version dans LOVEppears

QC : 3,5/4 ; QR : 3/4 ; Arr : 3,75/5 ; Str 0,5/1 ; Ch : 1,25/2 ; Par : 2/2 ; CC : 1,5/2   -> 15,5/20

 

LOVE ~since 1999~

Un bon duo (avec Tsunku), un peu répétitif, mais bien plus rythmée que les calmes LOVE~destiny~ et LOVE~refrain~. Sobre. 

QC : 3,5/4 ; QR : 3,25/4 ; Arr : 3,75/5 ; Str 0,5/1 ; Ch : 1,5/2 ; Par : 1,75/2 ; CC : 1,5/2   -> 15,75/20

 

ever free

La dernière piste du single vogue est très regrettable :

regrettable de ne pas faire partie de l’album Duty, avec lequel l’album aurait été sublime (ever free en dernière piste de Duty aurait été parfait).

On peut aussi regretter le fait que la chanson soit trop courte (juste un couplet puis deux refrains).

Car ever free a tout pour être sublime, avec une ambiance très apaisante…

QC : 3,75/4 ; QR : 3,5/4 ; Arr : 5/5 ; Str 0,5/1 ; Ch : 1,25/2 ; Par : 2/2 ; CC : 2/2   -> 18/20

 

Endless sorrow

Cette version originale n’est présente que sur le single du même nom et sur A BEST 2 -BLACK-

En effet, Endless sorrow est magnifiquement arrangée, un début et un final au piano, des guitares entraînant. Le pont de guitares et de violons frôlent l’excellence.

On peut encore se demander POURQUOI cette version n’est pas présente dans l’album I am…

QC : 3,5/4 ; QR : 3,75/4 ; Arr : 5/5 ; Str 0,75/1 ; Ch : 1,75/2 ; Par : 1,5/2 ; CC : 2/2   -> 18/20

 

RAINBOW

Présent dans l’album A BALLADS.

Ce qui suit est un copié-collé d’une précédente critique :

Une magnifique ballade, l’une des plus belles d’Ayu ; la structure en crescendo est magnifique. En effet, on a un premier couplet/refrain très reposant suivi d’un second avec des sonorités de guitares basses. Tout ceci, accompagné d’un second refrain, couplé avec des chœurs font de cette chanson, quelque chose de sublime !

« No rain, Can’t get the Rainbow »

QC : 4/4 ; QR : 4/4 ; Arr : 5/5 ; Str : 1/1 ; Ch : 1,25/2 ; Par : 1,75/2 ; CC : 2/2   -> 19/20

 

 

Sotsugyô shashin

Présent dans A BALLADS, sotsugyou shashin est une reprise de Yumi Matsutoya. Il ne s’agit pas de la meilleure ballade, mais elle est tout de même reposante et très sobre. En tout cas la version acoustique chantée lors du Countdown Live 2007-2008 est de meilleure qualité que la version CD.

QC : 3,75/4 ; QR : 3,25/4 ; Arr : 3,75/5 ; Str 0,5/1 ; Ch : 1,5/2 ; Par : 1,25/2 ; CC : 1,5/2   -> 15,5/20

 

theme of a-nation ‘03

Présent dans le single & et malheureusement pas dans le mini-album Memorial address, theme of a-nation ’03 était censé être chanté pendant le a-nation comme son nom l’indique, mais cela ne s’est pas fait finalement, laissant le titre n’ayant aucun rapport avec la chanson…

Concernant le morceau, on peut dire qu’elle est très contrastée. En effet, autant la première partie avec un style très chinois, est magnifique, autant la seconde, constituée de chœurs est répétitive et longue, trop longue (2 minutes). Ce qui est dommage, car la chanson avait le potentiel d’être vraiment magnifique dans la globalité.

QC : 3,5/4 ; QR : 3,5/4 ; Arr : 4,5/5 ; Str 0,5/1 ; Ch : 1,5/2 ; Par : 1,5/2 ; CC : 1,5/2   -> 16,5/20

 

teens

Cette reprise a un léger faux-air de Noël, avec un début que j’aime beaucoup (piano, puis « I wanna be a sweet angel », puis violons). Seulement, bien que très sympa, je trouve cette chanson trop répétitive…

QC : 3,25/4 ; QR : 3/4 ; Arr : 3,75/5 ; Str 0,5/1 ; Ch : 1,5/2 ; Par : 1,5/2 ; CC : 1,5/2   -> 15/20

 

part of Me

Une ballade mid-tempo assez sympa. Il faut voir le clip ou le live pour constater un cetrtain ton asiatique sur cette chanson qui se laisse facilement écouter, mêùe si elle n’est pas inoubliable pour autant.

QC : 3,25/4 ; QR : 3,25/4 ; Arr : 3,5/5 ; Str 0,5/1 ; Ch : 1,5/2 ; Par : 1,25/2 ; CC : 1,25/2   -> 15/20

 

 

 

 

BONUS :

 

kanariya “Jonathan Peters' Vocal Club Mix”

J’ai choisi de faire une mention spéciale à la piste principale du single kanariya, parce que je la trouve bien meilleure que la version album. C’est d’ailleurs cette version remixe qui a été utilisée pour le clip… En résumé, avec un meilleur rythme, et un style plutôt typé dance/techno, cette version aurait largement mérité sa place au milieu de l’album LOVEppears. Par ailleurs elle est vraiment très longue… Plus de 9 minutes…

QC : 3,5/4 ; QR : 3,5/4 ; Arr : 4,5/5 ; Str 1/1 ; Ch : 1/2 ; Par : 1,25/2 ; CC : 1,75/2   -> 16,5/20

 

A Song for XX “030213 Session #2”

Inclus dans A BALLADS, cette version de A Song for XX est de loin le meilleur réarrangement qu’à fait Ayu jusqu’à maintenant. Elle y chante mieux, pour l’instrumentalisation, rien à redire. Pour vous dire, j’aimais la version originale mais sans plus, jusqu’au moment où je découvre la version A BALLADS *o*. Je suis alors tombé amoureux de A Song for XX. Et comme dit précédemment, les paroles sont parmi les plus beaux écrits d’ayumi. Excellent.

QC : 3,75/4 ; QR : 4/4 ; Arr : 4,5/5 ; Str 1/1 ; Ch : 1,75/2 ; Par : 2/2 ; CC : 2/2   -> 19/20
